## 🚀 Featured Projects

### 🔍 AI Job Search Agent
> `FastAPI` `LangChain` `FAISS` `React` `SQLite` `SSE`

A multi-agent AI system that **processes 1000+ job listings** and delivers hyper-personalized recommendations based on resume analysis and semantic skill matching.

- 🎯 **85% relevance accuracy** via hybrid FAISS + LLM + rule-based ranking
- ⚡ Real-time streaming interface (SSE) showing agent reasoning — **40% lower perceived latency**
- 🧠 Built with LangChain agents for orchestrated multi-step reasoning

---

### 🌾 CropSense Karnataka
> `Python` `scikit-learn` `XGBoost` `Flask` `Pandas` `NumPy`

An ML-powered **crop recommendation system** trained on 5000+ real-world soil and weather data points from Karnataka.

- 📈 **89% prediction accuracy** across 5 crop types using Random Forest + XGBoost
- 🧪 Feature engineered on pH, NPK levels, rainfall, and temperature data
- 🗺️ Flask API with geo-location + seasonal awareness for real-time recommendations
